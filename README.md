# EduFlex Global Website

A premium student consultancy website with stunning animations and modern design.

## Features

- 🎨 **Mind-blowing animations** with Framer Motion
- 📱 **Mobile-optimized** responsive design
- 🌐 **3D effects** and parallax scrolling
- ⚡ **Performance optimized** with lazy loading
- 🎯 **Accessibility** focused with reduced motion support

## Technologies Used

- React (via Babel)
- Tailwind CSS
- Framer Motion
- Vanilla JavaScript

## Deployment

### Vercel (Recommended)
1. Go to https://vercel.com/new
2. Drag & drop this folder
3. Deploy instantly

### Netlify
1. Go to https://app.netlify.com/drop
2. Drag & drop this folder
3. Deploy instantly

## Local Development

```bash
# Start local server
python -m http.server 8080

# Or use Node.js
npx serve .
```

Visit `http://localhost:8080` to view the site.

## File Structure

```
project/
├── index.html              # Main entry point
├── EduFlex Global.html     # Original file
├── vercel.json            # Vercel configuration
├── .gitignore             # Git ignore rules
├── README.md              # This file
└── assets/                # Images and media
    ├── team/              # Team member photos
    ├── success/           # Success stories
    └── uploads/           # User uploaded images
```

## Customization

- Edit `index.html` to modify content
- Update CSS variables in `<head>` for brand colors
- Modify animations in the `<style>` section
